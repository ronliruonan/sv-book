语法
==================
```
  func.apply(thisArg, [argsArray])
```

参数
------------------
- thisArg
  - 当this为null 或者 undefined 时，自动替换为全局对象
- argsArray
  - 一个数组或者类数组对象， 如果该参数为null 或者 undefinded 表示不传入任何参数
