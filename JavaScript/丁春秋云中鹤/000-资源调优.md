延迟加载通过将应用程序拆分为多个包并按需加载它们来加快应用程序加载时间。要使用延迟加载，请提供loadChildren属性而不是children属性。
[{
   path: 'parent/:id',
   children: [
     { path: '', component: MainChild },
     { path: '', component: AuxChild, outlet: 'aux' }
   ]
}]

[{
  path: 'team/:id',
  component: Team,
  loadChildren: 'team'
}]
