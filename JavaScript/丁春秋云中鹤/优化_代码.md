
## 函数中经常用到的全局变量，把他保存在局部变量中
- 减少作用域查找
- 尽量避免使用全局变量，使用局部变量

```
var a = 10;
function cc(){
    console.log(a)
    // var temp = a;
    // console.log(temp);
}
cc();
```

## `for( ; ; )` 迭代
-  arr.length 提出来
-  递减方式：编译的指令少

```
var arr = [23,3,4,5,6,45];
var max = arr[0];
for(var i= 0; i< arr.length; i++){
    if(arr[i]>max){
        max=arr[i]
    }
}
```
