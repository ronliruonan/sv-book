

Math.abs(x)
--------------
返回x的绝对值

Math.ceil(x)
--------------
返回x向上取整后的值

Math.floor(x)
--------------
返回小于x的最大整数

Math.round(x)
--------------
返回四舍五入后的整数

Math.max([x,[,y[,...]]]) / Math.min([x,[,y[,...]]])
--------------
返回哦个多啊多个数值中最的那个。
```
Math.max(1,2,3,4,5,67,7,8)
```

Math.random()
--------------
返回0-1之间的伪随机数
