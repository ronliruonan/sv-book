## 传输层（TCP UDP ）
LVS  负载的一个东西

### TCP： Transmission Control Protocol
是基于连接的协议，也就是说在正式收发数据之前，必须建立可靠的连接

### UDP： User Data Protocol
面向非连接的协议

### TCP vs UDP
TCP  vs  UDP
连接     非连接
可靠     不可靠
数据大   数据少
慢       快
